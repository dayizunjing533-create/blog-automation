# /outline コマンド

このコマンドは、note記事の構成案（アウトライン）を生成する。

## 実行手順
1. CLAUDE.md を読み込む
2. ユーザーから渡された執筆指示（Pattern AまたはPattern B）を確認する
3. 執筆指示にメインKWが含まれていない場合は、執筆を開始せず確認を求める
4. 以下の形式で構成案を出力する

## 出力フォーマット
タイトル: （35文字以内、メインKWを前方に配置）

リード文: （50〜100文字、結論→メリット→要約の順）

H2-1: （QUEST法のQ: 問題提起） H3（必要な場合のみ）:

H2-2: （QUEST法のU: 共感） H3（必要な場合のみ）:

H2-3: （QUEST法のE: 解決策） H3（必要な場合のみ）:

H2-4: （QUEST法のS: 証拠） H3（必要な場合のみ）:

H2-5: （QUEST法のT: 行動喚起・まとめ）
## ルール
- H2は3〜5個。QUEST法の全要素を無理に入れなくてよい
- 各H2に「この見出しで伝えること」を1行で併記する
- 構成案を出力したら必ず人間の承認を待つ。承認なしで本文執筆に進まない
- 見出しの文言は結論が分かる表現にする
