# /outline コマンド

このコマンドは、note記事の構成案（アウトライン）を生成する。

## 実行手順
1. CLAUDE.md を読み込む
2. ユーザーから渡された執筆指示（Pattern AまたはPattern B）を確認する
3. 執筆指示にメインKWが含まれていない場合は、執筆を開始せず確認を求める
4. 以下の形式で構成案を出力する

## 出力フォーマット
タイトル: （35文字以内、メインKWを前方に配置）

リード文: （50〜100文字、結論→メリット→要約の順）

H2-1: （共感・ストーリー導入。読者の悩みを描写し、問題提起と経験提示で始める。PREP法は適用しない）
H3（必要な場合のみ）:

H2-2: （PREP法: 結論→理由→具体例→まとめ）
H3（必要な場合のみ）:

H2-3: （PREP法: 結論→理由→具体例→まとめ）
H3（必要な場合のみ）:

H2-4: （PREP法: 結論→理由→具体例→まとめ）
H3（必要な場合のみ）:

H2-5: （PREP法: 結論→理由→具体例→まとめ。行動喚起・まとめとして使う場合はこの構成で締める）

## ルール
- H2は3〜5個。全要素を無理に入れなくてよい
- 各H2に「この見出しで伝えること」を1行で併記する
- 構成案を出力したら必ず人間の承認を待つ。承認なしで本文執筆に進まない
- 見出しの文言は結論が分かる表現にする
