# NGワードリスト

## このファイルの目的
このブログの読者は「副業を1〜3回挫折した20代会社員」である。
親近感と信頼感の両立を求めており、説教臭い表現・誇張表現・AI特有の文体は
読者の離脱と不信感を招くため、以下を禁止する。
執筆・校閲時は必ずこのファイルを参照し、該当表現があればBefore/Afterの方針に従いリライトすること。

## NG表現一覧

### 文末・語尾系（説教感・他人事感）
- 〜と言えるでしょう
- 〜ではないでしょうか
- 〜することが重要です
- 〜することが大切です
- 〜を心がけましょう
- いかがでしたか
- ぜひ参考にしてみてください

### 誇張・曖昧系（信頼性を下げる）
- 非常に／とても（強調目的）
- 絶対に稼げる
- 誰でも簡単に
- すぐに結果が出る
- 〜すれば必ず

### AI臭ワード（機械的・無機質に聴こえる）
- 〜という点が挙げられます
- 〜と考えられます
- 本記事では〜について解説します
- 以上が〜についての解説です
- まとめると以下のとおりです（表の直前以外）
- 〜のポイントは以下の3つです（H2直下での一度限りはOK。乱用禁止）

### 成功者・上から目線系（ペルソナが拒絶する）
- 月収100万円も夢じゃない
- 努力すれば必ず報われる
- やる気さえあれば誰でもできる
- 成功するためには〜が必要です

## Before/After（リライト基準）

| NG例 | OK例 |
|---|---|
| 〜することが大切です | 僕はまず〜から始めました。これが一番効果的だったからです |
| 絶対に稼げる | 初月で8,000円の収益が発生しました |
| 誰でも簡単にできます | スキルゼロの僕でも、て2週間で初案件を取れました |
| 〜することが重要です | 実際にやってみてわかったのは、〜だということです |
| 非常に効果的です | 体感で2倍くらい速くなりました |
| いかがでしたか | （削除。まとめ表または次の記事への導線に置き換える） |

## 感情の扱い方针

### 禁止：綁濾事でまとめること
- 挫折・失敗・面倒くさ・悲しさを隠して前向きに締めない
- 「でも頭張れば大丈夫です」で逃げない

### 推奨：負の感情をそのまま書く
- 「正直しんどかったです」
- 「また失敗するかもと思っていました」
- 「面倒で2日放置しました」

### 方针
- 語り手は「僕」で統一する
- 読者の横に座っている同い年の先輩のような距離感を保つ
- 完璧な成功者ではなく、今まさに同じ場所にいる人間の記録として書く
