# セクション3: 内部リンク構造・トピッククラスタ設計

---

## トピッククラスタの全体像

メインピラー記事（1記事）
  → サブピラー記事（6記事：各カテゴリに1つ）
    → 個別記事（各カテゴリ10〜15記事）

すべての個別記事はサブピラーを経由してメインピラーに帰結する。

---

## 記事の種類と役割

| 種類 | 目的 | 文字数 | 記事数 |
|------|------|--------|--------|
| ピラー記事 | SEO権威集中・サイト全体のハブ | 3000〜5000字 | 1記事 |
| サブピラー | カテゴリ統括・配下記事へのハブ | 2500〜4000字 | 6記事 |
| 集客記事 | 検索流入獲得・ロングテールKW狙い | 2000〜3500字 | 50〜60記事 |
| 収益記事 | アフィリエイト成約 | 2500〜4000字 | 8〜12記事 |
| 共感記事 | SNS流入獲得・ファン化 | 1500〜2500字 | 5〜8記事 |

---

## 内部リンクルール

### 基本ルール
1. すべての個別記事 → 所属カテゴリのサブピラー記事へリンク
2. すべてのサブピラー記事 → メインピラー記事へリンク
3. メインピラー記事 → 全サブピラー記事へリンク
4. 同一カテゴリ内の関連記事同士は相互リンク可
5. 異なるカテゴリ間のリンクは文脈上自然な場合のみ

### リンクの貼り方
- 本文中の自然な文脈で貼る
- 「詳しくはこちら」は最小限にする
- 記事末尾に「あわせて読みたい」として2〜3記事を提示
- サイドバーやCTAでピラー記事へ常時導線を設置

### 禁止事項
- 1記事内に10本以上の内部リンクを貼らない
- 関連性のない記事への内部リンクは貼らない
- 同じリンク先を1記事内で3回以上貼らない

---

## 収益記事への導線パターン

### パターンA: 集客記事 → 収益記事
例: 「読書 続かない」の集客記事
→ 本文内で「耳で読む方法もある」と触れる
→ 「Audibleレビュー」収益記事へ内部リンク

### パターンB: 集客記事 → サブピラー → 収益記事
例: 「早起き 習慣化」の集客記事
→ サブピラー「朝活を続ける仕組み」
→ 「光目覚ましレビュー」収益記事へリンク

### パターンC: 共感記事 → ピラー → 回遊
例: SNSから「何か変わりたい」共感記事へ流入
→ ピラー記事「習慣化完全ガイド」
→ 各カテゴリの収益記事へ分岐
